welcome to my webserver

create by arfian with module express nodejs

how to use

  1. create a file in the htdocs folder
  2. this server can only run html files
  3. you can't put files in a folder in the htdocs folder
  4. I don't recommend creating separate css and js files, if forced then use http://[url.css/js] to link
  5. to access the file you can type in the url IP:3000/[name file+extention]
  6. example : http://192.168.1.5:3000/example.html
